# Решение шифра Цезаря

Даны пары открытый текст – шифротекст. Требуется найти ключ шифра Цезаря.

## Условие задачи

- а. АПЕЛЬСИН → САЦЬНВЩЮ  
- б. АБРИКОС → ЫЬЛГЕЙМ

## Подробное решение

### Используемый алфавит

Для расчётов используется русский алфавит из **32 букв** (без "Ё"):


### Часть (а): АПЕЛЬСИН → САЦЬНВЩЮ

Сравниваем позиции каждой буквы:

| Открытая | Индекс | Шифрованная | Индекс | Сдвиг |
|----------|--------|--------------|--------|-------|
| А        | 0      | С            | 17     | +17   |
| П        | 15     | А            | 0      | -15 ≡ +17 (mod 32) |
| ...      | ...    | ...          | ...    | ...   |

Все сдвиги дают **+17**.

✅ **Ключ = 17**

### Часть (b): АБРИКОС → ЫЬЛГЕЙМ

Аналогично:

| А → Ы: 0 → 27 → +27 |
| Б → Ь: 1 → 28 → +27 |
| Р → Л: 16 → 11 → -5 ≡ +27 (mod 32) |

✅ **Ключ = 27**

## Ответ

- а. Ключ шифра Цезаря: **17**
- б. Ключ шифра Цезаря: **27**

> Примечание: сдвиг на 27 вперёд эквивалентен сдвигу на 5 назад, так как 27 ≡ -5 (mod 32).
